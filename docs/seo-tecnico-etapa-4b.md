# SEO técnico base — Etapa 4B

## Alcance

Se aplicó SEO técnico base al sitio D&A Systems sin modificar diseño visual, estructura Bootstrap, CSS, JS, imágenes ni formulario.

## Páginas modificadas

- index.html
- servicios-ti.html
- soluciones-ti.html
- nosotros.html
- terminos-condiciones.html
- politica-privacidad.html

## Cambios aplicados

- Meta description única por página.
- Eliminación de meta keywords vacía.
- Canonical por página.
- Open Graph básico:
  - og:title
  - og:description
  - og:type
  - og:url
  - og:image
- robots.txt actualizado.
- sitemap.xml creado con las páginas principales.

## Dominio usado

Se usó como dominio objetivo:

https://www.dasystems.cl/

Antes de publicar, confirmar que este será el dominio definitivo del sitio.

## Archivos de publicación

- robots.txt
- sitemap.xml

## Revisión recomendada antes de publicar

- Confirmar dominio definitivo.
- Confirmar que el sitio cargue por HTTPS.
- Probar URLs canónicas.
- Validar sitemap.xml.
- Validar Open Graph con una herramienta de previsualización.
